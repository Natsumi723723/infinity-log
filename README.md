# ∞ INFINITY LOG

**Personal timeline diary** — 画面遷移ゼロ、1画面で完結する日記アプリ。

🔗 **https://natsumi723723.github.io/infinity-log/**

---

## Features

- **自動保存** — 入力を止めると自動保存。送信ボタンなし
- **タイムライン形式** — 最新が上、日付で自動グループ化
- **タップで編集** — 過去のメモをタップするだけで編集モード
- **クラウド同期** — Supabase でスマホ↔PC リアルタイム同期
- **ダーク / ライトテーマ** — ☀️🌙 ボタンで切り替え、設定を記憶
- **PWA対応** — ホーム画面に追加してアプリ感覚で使える

## Tech

- React 18 (CDN / Babel standalone)
- Tailwind CSS (CDN)
- Supabase (Auth + Realtime DB)
- GitHub Pages

## Design

Cyber Maximalism — ネオンピンク × ダークサイバーパープル
